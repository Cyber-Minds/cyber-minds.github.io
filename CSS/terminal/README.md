# Terminal Frontend CSS Modules

`CSS/terminal.css` is the public entrypoint and imports these modules:

- `base.css`
- `shell.css`
- `challenge.css`
- `editor.css`
- `terminal-panel.css`
- `overlay.css`
- `responsive.css`
- `scrollbar.css`

Keep selectors grouped by concern to avoid regressions during UI updates.
