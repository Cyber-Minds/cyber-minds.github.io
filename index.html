    Cybersecurity Opportunities

    "Cybersecurity is not just about protecting data, but about protecting the trust in our digital world."

    - The CyberMinds Board

    Information Security Analyst

    Average Salary: $99,274

    Protect an organization's computer networks and systems by:

    • Monitoring networks for data security vulnerabilities
    • Investigating, documenting, and reporting security breaches
    • Researching IT security trends
    • Helping computer users with security products and procedures
    • Developing strategies to help their organization remain secure

    Digital Forensic Examiner

    Average Salary: $96,157

    Digital forensic investigators retrieve information from computers and other digital devices to discover how an unauthorized person accessed a system or to gather evidence for legal purposes. Day-to-day tasks might include:

    • Collecting, preserving, and analyzing digital evidence
    • Recovering data from erased or damaged hard drives
    • Documenting the data retrieval process and maintaining chain of custody
    • Assisting law enforcement in criminal investigations
    • Providing expert testimony in court proceedings

    Security Systems Administrator

    Average Salary: $114,967

    Manages the day-to-day operations of an organization's cybersecurity systems, ensuring smooth and secure performance.

    • Monitoring systems and running regular backups
    • Managing individual user accounts
    • Developing and documenting security procedures
    • Collaborating with security teams to respond to intrusions
    • Participating in company-wide security audits

    Penetration Tester

    Average Salary: $124,127

    Helps businesses identify security weaknesses by simulating attacks on their computer networks.

    • Planning, designing, and carrying out penetration tests
    • Creating reports and offering recommendations
    • Developing scripts to automate testing
    • Conducting social engineering exercises
    • Providing technical support during incident handling

    Security Engineer

    Average Salary: $137,879

    Designs systems to protect a company’s computers, networks, and data from cyberattacks and disasters.

    • Developing security standards and best practices
    • Recommending security enhancements to management
    • Ensuring new security systems are installed correctly
    • Testing security solutions
    • Leading incident response teams
    • Developing programs to automate vulnerability detection

    Security Architect

    Average Salary: $164,433

    Sets the vision for a company’s security systems, identifying vulnerabilities and building strategies to strengthen security across networks and systems.

    • Building and maintaining security networks and systems
    • Preparing budgets and overseeing security expenses
    • Coordinating security operations across IT and engineering departments
    • Improving systems in response to security incidents or vulnerabilities
    • Conducting breach of security drills

    Cryptography Engineer

    Average Salary: $137,802

    Secures data for communication and information exchange by creating encryption algorithms, ciphers, and security protocols, and analyzing coded information.

    • Developing new cryptographic algorithms
    • Analyzing existing algorithms for vulnerabilities
    • Implementing encryption solutions
    • Testing new encryption techniques and tools

    Cybersecurity Manager

    Average Salary: $153,084

    Oversees the security infrastructure of an organization, managing resources, policies, and risk mitigation efforts to ensure compliance and protection.

    • Managing human and technological resources
    • Tracking changes to internal and external security policy
    • Ensuring compliance with security rules and regulations
    • Sourcing cybersecurity tools for the organization
    • Leading risk mitigation efforts
