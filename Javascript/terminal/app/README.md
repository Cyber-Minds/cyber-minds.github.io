# Terminal App Runtime Modules

- `core.js`: state persistence, palette/theme controls, editor tab switching.
- `workspace.js`: workspace file reads/sync and terminal resize messaging.
- `challenges.js`: challenge loading, completion checks, and progress updates.
- `runtime.js`: panel resizing, terminal theme, and terminal session lifecycle.
- `ui.js`: keyboard shortcuts, UI event binding, Monaco setup, and bootstrap.

Loader entrypoint: `../app.js`.
